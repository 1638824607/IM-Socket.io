##代码演示

server

```plain
$ cd srever
$ npm install 
$ node index.js
```

client
```plain
$ cd client
$ npm install 
$ electron .
```


##实现功能
- 登陆页面进入用户聊表页面，即时添加用户人数
- 点击列表内用户进入聊天页面,实现简单聊天功能
